# Cricket-Dashboard
An interactive Power BI dashboard that analyzes cricket match data to visualize player and team performance, track key metrics like runs and wickets, and identify winning trends. It helps users gain actionable insights and make data-driven decisions through clear, dynamic visualizations.

# 🏏 Cricket Champions Trophy Dashboard

A Power BI dashboard analyzing ICC Champions Trophy match results and player statistics.

## 📁 Files
- `CA_prac_DB.pbix` — Main Power BI dashboard
- `all_champions_trophy_matches_results.csv` — Match results data (116 records)
- `all_champions_trophy_players_list.csv` — Player list data (1036 records)
- `team_flags.csv` — Team flag image URLs

## 🛠 Tools Used
- Power BI Desktop
- Microsoft Excel / CSV

## 📊 Features
- Match results by team and year
- Player of the Match analysis
- Win/Loss ratios and rankings
- Team flag visuals for 13 nations
